# Calculator

A clean, modern web-based calculator built with vanilla HTML, CSS, and JavaScript. No frameworks or dependencies required.

![Calculator](https://img.shields.io/badge/HTML-CSS-JavaScript-blue)

## Features

- **Basic operations** — Addition, subtraction, multiplication, division
- **Percent** — Calculate percentages
- **Decimal support** — Work with decimal numbers
- **Clear & Backspace** — Reset or remove last digit
- **Keyboard support** — Full keyboard shortcuts for faster input
- **Dark theme UI** — Sleek, minimal design with smooth interactions

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Number input |
| `.` | Decimal point |
| `+` `-` `*` `/` | Operators |
| `Enter` | Equals |
| `Backspace` | Delete last digit |
| `Escape` | Clear |

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Use the on-screen buttons or your keyboard to perform calculations

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
